# Примеры использования IMCS

Эта папка содержит примеры разреженных изображений для демонстрации работы IMCS.

## Тестовые изображения (16×16)

Все изображения специально подобраны для демонстрации Compressed Sensing.  
**Ключевой фактор**: разреженность в DCT базисе.

### Отлично работает (< 10% значимых DCT коэффициентов)

| Файл | Описание | Разреженность | PSNR (ratio=0.5) |
|------|----------|--------------|------------------|
| `01_constant.png` | Константное (все пиксели = 150) | 0.4% | **314 dB** |
| `02_gradient_h.png` | Горизонтальный градиент | 1.6% | **301 dB** |
| `03_gradient_v.png` | Вертикальный градиент | 1.6% | **294 dB** |
| `04_square.png` | Квадрат в центре | 9.8% | **303 dB** |
| `06_stripes_v.png` | Вертикальные полосы | 3.5% | **297 dB** |
| `07_stripes_h.png` | Горизонтальные полосы | 3.5% | **298 dB** |

**Результат:** Почти идеальное восстановление.

### Хорошо работает (10-30%)

| Файл | Описание | Разреженность | PSNR (ratio=0.5) |
|------|----------|--------------|------------------|
| `08_checkerboard.png` | Шахматная доска 4×4 | 20.3% | **56 dB** |
| `09_diagonal.png` | Диагональная линия | 24.6% | **26 dB** |

**Результат:** Хорошее качество, небольшие артефакты.

### Приемлемо работает (>30%)

| Файл | Описание | Разреженность | PSNR (ratio=0.5) |
|------|----------|--------------|------------------|
| `05_circle.png` | Круг в центре | 53.9% | **16 dB** |
| `10_cross.png` | Крест | 39.1% | **23 dB** |

**Результат:** Видны артефакты, но изображение узнаваемо.

## Использование

### Обработать лучшие примеры

```bash
python main.py --input 01_constant.png --ratio 0.5

python main.py --input 02_gradient_h.png --ratio 0.3

python main.py --input 04_square.png --ratio 0.4

python main.py --input 06_stripes_v.png --ratio 0.5
```

### Обработать все изображения сразу

```bash
python main.py --ratio 0.5 --algorithm omp
```

### Результаты

После обработки смотрите:
- `examples/output/<название>/original.png` - исходное
- `examples/output/<название>/reconstructed_omp.png` - восстановленное
- `examples/output/<название>/report.txt` - метрики
